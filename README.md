# Angular_SwApi

Angular project that display Star Wars films data, making request to [SwApi](https://swapi.co/api/films/)

## Getting Started

Run inside the project folder:
```
npm i
```

to run the app type:
```
ng serve
```

### Prerequisites

Install [node](https://nodejs.org/it/download/)
